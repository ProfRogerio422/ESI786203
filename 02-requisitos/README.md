# Unidade 2 – Requisitos de Software

Objetivo: Aprender a identificar, documentar e gerenciar requisitos de software.

## 📚 Conteúdos
- Tipos de requisitos (funcionais e não funcionais)
- Técnicas de elicitação (entrevistas, questionários, observação)
- Especificação de requisitos
- Validação e gerenciamento de requisitos
