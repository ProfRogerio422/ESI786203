# ESI786203
Este repositório reúne materiais, exemplos e recursos da disciplina Engenharia de Software I a Objetos (ESI786203). O objetivo é apoiar o aprendizado dos fundamentos da engenharia de software com foco em técnicas e práticas orientadas a objetos.

Conteúdos abordados

    Fundamentos da engenharia de software

    Processos de Software

    Análise e especificação de requisitos
    
    Modelagem orientada a objetos (UML)

    Principios de Projeto   

    Introdução a testes e qualidade de software
